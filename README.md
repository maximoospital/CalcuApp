## Run locally

You will need node and npm installed in your computer to run this app.

Clone the project

```bash
  git clone https://github.com/maximoospital/calculapp.git
```

Go to the project directory

```bash
  cd calculapp
```

Install requirements

```bash
  npm installl
```
Run app

```bash
  npm start
```

Visit app

```bash
  localhost:3000
```
